# MeshCore

## Guia de configuração
- [Configuração Recomendada](configuracao.md): Como configurar seu dispositivo MeshCore para operar na Mesh Sorocaba.
- [Como Configurar um Observer no MeshCore](como-configurar-um-observer-no-meshcore.md): Configure um dispositivo como `Observer` para monitorar a rede e contribuir com a comunidade.
- [Gerador de Nomes para Repetidoras](nomes-repetidoras.md): Auxilia na escolha de um nome para sua repetidora MeshCore.

## Mapas e ferramentas
- [map.meshcore.io](https://map.meshcore.io/?zoom=10&lat=-23.4607&lon=-47.5763): Mapa oficial do MeshCore.
- [Let's Mesh](https://analyzer.letsmesh.net/packets?region=SOD): Analizador de tráfego em tempo real para a região de Sorocaba.
- [MeshMapper](https://sod.meshmapper.net/): Mapeamento da cobertura do MeshCore na região de Sorocaba; dados coletados por voluntários pelo aplicativo [MeshMapper](https://play.google.com/store/apps/details?id=net.meshmapper.app&hl=pt_BR).
